# dotfiles

[chezmoi](https://github.com/twpayne/chezmoi) で管理。

## 使い方

```bash
$ brew install chezmoi
$ chezmoi init git@github.com:kk6/dotfiles.git
$ chezmoi apply
```

chezmoi は Win でも Linux でも使えるので Mac 以外は公式の使い方参照。
